Crime statistics in each day

The graph shows a bar chart called a "frequency graph" that visualizes the number of crimes committed on different days of the week. Each bar in the graph represents the number of crimes on a particular day, with the number of days in a week labeled on the x-axis. There are three sets of bars, each corresponding to a different category, and each category is distinguished by color. The red bar represents the "Total Count", which indicates the total number of crimes recorded for each day. The light blue bars represent "Count in SE," showing the crime counts for Seattle. The green bar represents "Count in NY" referring to New York.
